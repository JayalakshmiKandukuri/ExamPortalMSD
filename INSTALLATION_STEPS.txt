========================================
ONLINE EXAM SYSTEM - INSTALLATION STEPS
========================================

STEP 1: INSTALL DEPENDENCIES
-----------------------------

1. Open PowerShell/Command Prompt in the ProjectMSD folder

2. Install Backend Dependencies:
   cd backend
   npm install
   cd ..

3. Install Frontend Dependencies:
   cd frontend
   npm install
   cd ..


STEP 2: SETUP ENVIRONMENT VARIABLES
------------------------------------

1. Navigate to the backend folder
2. Copy the .env.example file and rename it to .env
3. Edit the .env file with these values:

   PORT=5000
   MONGODB_URI=mongodb://localhost:27017/online-exam-system
   JWT_SECRET=your_secret_key_here_change_this_12345
   NODE_ENV=development

   (You can use any random string for JWT_SECRET)


STEP 3: START MONGODB
----------------------

Option A - Local MongoDB:
   - Make sure MongoDB is installed on your system
   - Start MongoDB service or run: mongod

Option B - MongoDB Atlas (Cloud):
   - Create free account at https://www.mongodb.com/cloud/atlas
   - Create a cluster
   - Get connection string
   - Replace MONGODB_URI in .env file


STEP 4: RUN THE APPLICATION
----------------------------

1. Open TWO separate terminal windows

2. Terminal 1 - Start Backend:
   cd backend
   npm run dev
   
   Wait for message: "Server running on port 5000" and "MongoDB Connected"

3. Terminal 2 - Start Frontend:
   cd frontend
   npm start
   
   Browser will open automatically at http://localhost:3000


STEP 5: FIRST TIME SETUP
-------------------------

1. Register Admin Account:
   - Go to http://localhost:3000/register
   - Enter your details
   - Select "Admin" as role
   - Click Register

2. Add Questions:
   - Login as admin
   - Click "Manage Questions"
   - Add multiple choice questions (minimum 5-10 questions)

3. Create Exam:
   - Go to admin dashboard
   - Click "Create Exam"
   - Fill exam details
   - Select questions
   - Set dates and time
   - Click "Create Exam"

4. Test as Student:
   - Logout
   - Register new account with "Student" role
   - Login and take exam
   - View results


QUICK COMMANDS REFERENCE
-------------------------

Backend:
  cd backend
  npm install          # Install dependencies
  npm run dev          # Start development server
  npm start            # Start production server

Frontend:
  cd frontend
  npm install          # Install dependencies
  npm start            # Start development server
  npm run build        # Build for production


TROUBLESHOOTING
---------------

Problem: MongoDB connection error
Solution: Make sure MongoDB is running, check connection string

Problem: Port already in use
Solution: Close other apps or change port in .env

Problem: Cannot see questions/exams
Solution: Make sure you're logged in with correct role (admin/student)

Problem: npm install fails
Solution: Delete node_modules folder and package-lock.json, then run npm install again


SYSTEM REQUIREMENTS
-------------------

- Node.js v14 or higher
- MongoDB v4.0 or higher
- Modern web browser (Chrome, Firefox, Edge)
- 4GB RAM minimum
- Internet connection (for npm packages)


DEFAULT PORTS
-------------

Backend API: http://localhost:5000
Frontend App: http://localhost:3000
MongoDB: mongodb://localhost:27017
